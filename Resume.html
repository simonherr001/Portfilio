Web Application Development

Web Application Development is a lab that I am currently taking at CART. This is a class where I learn to create websites from scratch using JavaScript, HTML, and CSS. Javascript, HTML, and CSS are the coding languages used to create these websites. Javascript is basically the one that makes things function correctly. With JavaScript only you can make a clock, timer, calculator and more. Next is HTML or HyperText Markup Language, this is where all the headers and text go to. This is where you write all your titles and paragraphs and when you do write the text in HTML it will pop up on your website. Lastly Is CSS or Cascading Style Sheets, this is where you can change everything visually. You can change text font, change the size of the font or a picture, or change the colors of things. There are many things you can do with CSS, how I like to think about this is that it makes everything look beautiful.
